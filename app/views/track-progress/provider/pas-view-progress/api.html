Get started with our APIs

What our APIs do

APIs (Application Programming Interfaces) let applications share information and data with each other.

We have three APIs you can use:

  • Recruitment API - this allows you to create adverts on Find an apprenticeship, using your own systems. If you want to create vacancies on employers' behalf using the API, you will need their permission. They can do this in their Apprenticeship service account
  • Display adverts API - this allows you to display adverts from Find an apprenticeship on your own websites
  • Track portable flexi job apprenticeship progress API - this allows us to access data on the progress of your portable flexi-job apprenticeships. This API is only for providers delivering these apprenticeships as part of a pilot

Setting up the APIs

You will need someone to connect your system with the APIs. As this is a technical process, the person best equipped is likely a software developer.

Share this page with the developer who is going to connect your system with the API. They will be able to try out the API using the sandbox provided..

Finding the API keys

You will need to give your developer API keys. The keys give your developer permission to connect to the API on your behalf.

If you have owner permissions you will be able to access the API keys here.
