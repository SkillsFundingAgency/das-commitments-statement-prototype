Using file upload

Before you start

There is a full specification on uploading new apprentices using a CSV file on GOV.UK.

You will need to know the employer agreement ID, this can be found on your Organisations and agreements page.

The CSV file is limited to 100 rows.

How it works

  • use file upload to add 1 or more apprentices for 1 or more employers
  • create new cohorts by leaving the 'CohortRef' field empty; the cohort reference will be created when you save or approve cohorts (provided you have the correct employer permissions)
  • make reservations automatically (provided you have the correct employer permissions)
  • add apprentices to existing cohorts that have not yet been approved by the employer
  • check for errors

Creating a CSV file

You can:

  • use your learner management system (LMS) to create a CSV file
  • download our template if you do not use a LMS
